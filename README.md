# Msg-XSI

[![Crate][crate-badge]][crate] [![docs-badge][]][docs] [![license-badge][]][license] [![travis-badge][]][travis] [![circle-badge][]][circle]

#### How to build:
```shell
git clone https://github.com/adjivas/msg.git msgxsi && cd msgxsi
cargo build
```

#### Directory-Tree:
```shell
.
|__ Cargo.toml
|__ LICENSE
|__ README.md
|__ tests
|   \__ lib.rs
|__ examples
|   |__ msg.rs
|   \__ clr.rs
\__ src
    |__ ffi.rs
    |__ lib.rs
    |__ macros.rs
    \__ main.rs
```
